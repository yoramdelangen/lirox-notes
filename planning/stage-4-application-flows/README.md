# Stage 4: Application Flows

This stage defines how users and systems move through LiroxNotes.

## Files

- `USER_FLOWS.md` contains user-facing flows.
- `SYNC_FLOWS.md` contains sync and Git gateway flows.
- `CONFLICT_FLOWS.md` contains conflict resolution flows.
- `ERROR_FLOWS.md` contains failure and recovery flows.
- `DESIGN_FLOWS.md` contains workflow for turning design direction into screen behavior.
