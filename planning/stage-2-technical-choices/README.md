# Stage 2: Technical Choices And Dependencies

This stage defines how LiroxNotes should be built technically.

## Topics

- Programming language.
- Frontend framework.
- Backend or Git gateway framework.
- PWA strategy.
- Git integration approach.
- Editor dependency.
- Markdown parsing and rendering.
- Search dependency.
- Local cache and offline storage.
- Auth and security.
- Testing tools.
- Build tooling.
- Deployment target.

## Files

- `TECHNICAL_CHALLENGES.md` contains implementation-specific questions collected during Stage 1.
- `SECURITY.md` contains security concerns, credential questions, encryption considerations, and gateway safeguards.
- `CHOICES.md` contains confirmed technical choices.
- `DEPENDENCIES.md` contains selected dependencies and why they were chosen.
