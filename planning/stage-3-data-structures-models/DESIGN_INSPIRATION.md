# Design Inspiration

## Visual Direction

- Minimalist.
- Calm.
- Fast.
- Clean.
- Low chrome.
- Writing-focused.
- File-tree driven.

## Product Feel

- Feels like a serious notes workspace, not a dashboard.
- Feels practical and lightweight.
- Prioritizes focus and readability.
- Avoids decorative clutter.

## UI Principles

- Keep the editor central.
- Keep navigation obvious.
- Keep labels and conflicts visible without noise.
- Prefer simple sidebars and clear states.
- Use spacing and hierarchy instead of ornament.

## Inspiration Notes

- Think simple note editor plus controlled file workspace.
- Think clean tools, not a complex productivity suite.
- Think direct, readable, calm, and functional.
